# Hi, I'm Ximi

I build backend systems and developer tools around LLM APIs.

My recent work focuses on turning multiple model providers into one manageable service: compatible APIs, routing, billing, quotas, observability, and repeatable container deployments.

## What I work with

- **Backend:** Java, Spring Boot, MyBatis-Plus, MySQL, Redis
- **Frontend:** Vue 3, TypeScript, Vite
- **Infrastructure:** Docker, Cloudflare Tunnel, Linux, GitHub
- **Current focus:** LLM gateways, API security, usage-based billing, deployment automation

## Featured project

### [Ximi LLM API](https://github.com/ximimoryan/ximi-llm-api)

A multi-provider LLM API gateway and operations console with OpenAI and Anthropic compatible endpoints.

- Provider accounts, model mapping, group routing and streaming responses
- API keys, wallets, subscriptions, quotas and token-based billing
- Usage logs, operational dashboards and gateway access controls
- Java 17 + Spring Boot 3.5 + Vue 3 + MySQL 8 + Redis 7
- Docker-based build, release, health check and rollback workflow
